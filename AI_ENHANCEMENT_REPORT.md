# 困难模式 AI 增强报告

## 📊 问题分析总结

根据详细分析，困难模式 AI 表现弱的主要原因：

### 核心问题

1. **搜索深度太浅**：3-4 层 → 应该 6-8 层
2. **思考时间太短**：800ms → 应该 2-3 秒
3. **候选位置太少**：20 个 → 应该 40-50 个
4. **评估函数简单**：缺少杀棋识别
5. **开局库简陋**：只有 3 种定式

// 连接奖励
let friendlyAdjacent = 0
for (let dr = -1; dr <= 1; dr++) {
  for (let dc = -1; dc <= 1; dc++) {

  }

score += friendlyAdjacent * PATTERNS.CONNECT_BONUS  // 每个 300 分
