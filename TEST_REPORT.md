// 使用 React.memo 优化单元格组件

### 2. 可访问性 ⚠️

#### 2.1 键盘导航
**问题**: 缺少键盘操作支持

**建议**:
```typescript
// 添加键盘导航
useEffect(() => {
  const handleKeyPress = (e: KeyboardEvent) => {
    if (e.key === 'ArrowUp' && cursor.row > 0) {
      setCursor(prev => ({ ...prev, row: prev.row - 1 }))
    }
    // ... 其他方向键
    if (e.key === 'Enter') {
      makeMove(cursor.row, cursor.col)
    }
  }
  window.addEventListener('keydown', handleKeyPress)
  return () => window.removeEventListener('keydown', handleKeyPress)
}, [cursor, makeMove])
```
