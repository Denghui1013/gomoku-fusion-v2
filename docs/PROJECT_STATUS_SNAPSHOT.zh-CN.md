## 1. 项目目标与范围

本项目是五子棋融合版独立工程，目标是形成可独立构建、可独立部署、可双端联机体验的完整产品形态，覆盖：

- 人机对战
- 联机对战（房间、实时对局、聊天、再来一局）
- 排位与奖励
- 统一移动端 UI 与结算体验

## 6. 关键交接提示（给下一位接手者）

1. 联机核心改动优先查看：
   - `src/hooks/useMultiplayer.ts`
   - `src/components/multiplayer/MultiplayerGameBackup.tsx`

2. 打包与 Android 相关优先查看：
   - `build-apk.bat`
   - `build-apk-versioned.bat`
   - `android/gradle/wrapper/gradle-wrapper.properties`
   - `android/gradle.properties`
3. 部署时优先遵循：
   - `DEPLOY_STANDALONE_VPS.md`
4. 若线上“看起来像旧版”或资源 500，优先排查顺序：
   - PM2 当前进程与端口是否唯一
   - Nginx `proxy_pass` 指向是否正确（3010）
   - `.next` 是否为最新构建产物
   - 浏览器/IAB 缓存是否已强制刷新
