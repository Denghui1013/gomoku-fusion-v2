## 1. 项目目标与范围

本项目是五子棋融合版独立工程，目标是形成可独立构建、可独立部署、可双端联机体验的产品版本，覆盖：

- 人机对战
- 联机对战（建房、加房、实时对局、聊天、悔棋、再来一局、认输、结算）
- 排位与奖励
- 统一移动端 UI 与交互反馈
